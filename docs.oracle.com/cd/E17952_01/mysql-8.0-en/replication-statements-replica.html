This section discusses statements for managing replica servers. Section 13.4.1, “SQL Statements for Controlling Source Servers”, discusses statements for managing source servers.

In addition to the statements described here, SHOW REPLICA STATUS and SHOW RELAYLOG EVENTS are also used with replicas. For information about these statements, see Section 13.7.7.35, “SHOW REPLICA STATUS Statement”, and Section 13.7.7.32, “SHOW RELAYLOG EVENTS Statement”.
