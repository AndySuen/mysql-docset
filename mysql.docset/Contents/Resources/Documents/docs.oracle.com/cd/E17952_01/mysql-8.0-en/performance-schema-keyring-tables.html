27.12.18 Performance Schema Keyring Tables

  • 27.12.18.1 The keyring_component_status Table
  • 27.12.18.2 The keyring_keys table

The following sections describe the Performance Schema tables associated with the MySQL keyring (see Section 6.4.4, “The MySQL Keyring”). They provide information about keyring operation:

  • keyring_component_status: Information about the keyring component in use.

  • keyring_keys: Metadata for keys in the MySQL keyring.
