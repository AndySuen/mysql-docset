32.3 Configuring Network Access

For information on OCI Security Rules, see Security Rules.

Important

You must enable ingress on the following ports:

  • 22: SSH

  • 3306: MySQL

  • 33060: (optional) MySQL X Protocol. Used by MySQL Shell.
